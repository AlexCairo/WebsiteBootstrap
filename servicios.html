DESARROLLO DE SOFTWARE DE VIDEOJUEGOS

Los desarrolladores están capacitados para usar Unreal Engine, CRYENGINE, Unity 3D / 5.0, Riot Engine, Maya , Houdini y más, además de JavaScript, HTML5, C / C ++ y más para modelar / renderizar animaciones.

PROGRAMACION DE SOFTWARE DE CAPTURA DE MOVIMIENTO

Programe archivos de captura de video de movimiento completo (mo-cap) y jerarquía de biovisión (BVH), además de software de reconocimiento de gestos, sensores integrados y varias funciones de la cámara.

Soluciones de Videojuegos 2d y 3d mind blast

Diseñe gráficos y animaciones 2D y 3D personalizados , así como juegos isométricos 2.5D de alta calidad, historias originales, niveles, personajes, física, lógica y efectos de sonido.

Soluciones de Software de Animación de Videojuegos

Los desarrolladores aprovechan los motores de juegos populares para modelar, texturizar, renderizar, manipular y animar activos de juegos para crear simulaciones y animaciones de juegos realistas.
